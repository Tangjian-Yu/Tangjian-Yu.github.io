# GitHub 仓库简介：tangjianYu.github.io

这是我的个人博客托管仓库，基于 Hexo 静态博客框架构建，通过 GitHub Pages 服务自动部署。

## 技术架构
- **框架**: Hexo 静态网站生成器
- **部署**: GitHub Pages 自动构建
- **写作**: Markdown 格式支持
- **访问**: https://Tangjian-Yu.github.io

## 仓库特点
- 使用 Git 进行版本控制，支持 SSH key 免密推送
- 基于 Node.js 环境，Hexo 命令快速生成静态页面
- 响应式设计，适配多设备访问

## 内容涵盖
- 技术学习笔记与心得分享
- 项目开发经验总结
- 编程技巧与踩坑记录
- 个人技术成长轨迹

按照教程配置，本仓库实现了从本地编写到自动部署的完整博客工作流。欢迎访问我的博客站点，如有建议欢迎通过 Issues 交流！
